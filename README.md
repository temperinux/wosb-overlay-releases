# 🚀 WoSB Overlay - Official Releases

Welcome to the official distribution hub for the **WoSB Overlay**! This tool is a high-performance, transparent Electron overlay designed for *World of Sea Battle* players to track guild standings (Enemies/Allies) in real-time.

## 📥 Download

Go to the [**Latest Release**](https://github.com/temperinux/wosb-overlay-releases/releases/latest) to download the app.

We provide two formats:
- **`WoSB Overlay Setup 1.0.0.exe` (Installer)**: Standard Windows installer (adds to Start Menu/Desktop).
- **`WoSB Overlay-1.0.0-win.zip` (Portable)**: No installation. Just unzip and run.

---

## ✨ Features
- **Real-Time Sync**: Instant Enemy/Ally list synchronization via Firebase.
- **Context-Aware Transparency**: Stays on top but becomes "click-through" when not in use.
- **Dual Layouts**: Switch between **Horizontal** and **Vertical** modes seamlessly.
- **Global Hotkeys**: Control the UI while your game is focused.
- **Member & Official Support**: Secure passcode access for members; admin codes for leaders.

---

## ⌨️ Essential Shortcuts
| Hotkey | Action |
| :--- | :--- |
| **`Alt + S`** | Toggle Overlay Visibility |
| **`Alt + F`** | Quick Search (Focuses the search bar) |
| **`Alt + D`** | **Reposition Mode** (Drag to your preferred screen corner) |
| **`Alt + V`** | Switch Layout (Vertical ↔ Horizontal) |
| **`Alt + Q`** | Session Menu (Logout or Close App) |

---

## 🛡️ Privacy & Security
- **No Personal Data**: The app only uses specialized Guild Passcodes.
- **Secure Backend**: Data is stored securely on Google Firebase.
- **Clean Registry**: The installer is light and removes cleanly.
